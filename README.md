# 📊 Prime Video Dashboard (Power BI)

## 📌 Overview
This project is an interactive **Power BI dashboard** built to analyze and visualize content available on **Prime Video**. It provides insights into shows, ratings, categories, countries, and trends over time.

The dashboard is designed to help users quickly explore patterns in streaming content and understand how movies and TV shows are distributed across different dimensions.

---

## 🎯 Key Features

### 🔢 High-level KPIs
- Total Categories: **519**
- Total Ratings: **25**
- Total Shows: **9,687**
- First Show Year: **1920**
- Last Show Year: **2021**

### ⭐ Ratings Analysis
- Distribution of shows by rating (e.g., 13+, 16+, 18+, PG-13)
- Helps identify content targeting different age groups

### 🎭 Category Insights
- Top genres:
  - Drama
  - Comedy
  - Drama & Suspense
- Understand content popularity by category

### 🌍 Geographical Distribution
- World map showing content production by country

### 🎬 Content Type Breakdown
- Movies: **~80.8%**
- TV Shows: **~19.2%**

### 📈 Trend Over Time
- Growth of movies and TV shows by year
- Strong increase in content after 2000

## 🚀 How to Use
1. Open the `.pbix` file using Power BI Desktop  
2. Use filters to explore:
   - Categories
   - Ratings
   - Years
3. Interact with visuals to gain insights

## 📈 Key Insights
- Majority of content is **Movies**
- **Drama** is the most dominant category
- Content production increased significantly after **2000**
- Content is globally distributed across many countries
